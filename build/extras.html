Here you'll find some external helpers and supplements that are not part of the core application.

An addition to the OS X service menu to pass the current Finder selection to Servus.

Alfred 2 workflow

A workflow for the beloved app launcher / productivity tool Alfred 2 that enables you to share a file or folder via Servus right from within the tool.

Servus for Mac Servus for Mac A Dropbox-powered sharing app for Mac. It's a menubar icon. Drop a file on it, and it'll build a preview page for it (custom themes!), upload it to your Dropbox, and copy the link for you.
